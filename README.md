# shopping-list
«Менеджер списка покупок — это удобное PWA-приложение, позволяющее создавать, сортировать и редактировать списки покупок, устанавливать напоминания, сохранять данные локально, экспортировать их в TXT/JSON и использовать встроенный калькулятор стоимости. Работает офлайн, поддерживает тёмную тему и может устанавливаться как мобильное приложение.»
